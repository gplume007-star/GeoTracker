The files contained in this export are TAB delimited and can be imported/viewed using Microsoft Excel, Microsoft Access or any other program that can access TAB delimited data.

NOTE: Some of the data files contained in this export contain more than 65536 records. Microsoft Excel 2003 and earlier versions cannot display more than 65536 records. It may be necessary to use Microsoft Access or Microsoft Excel 2007 (or newer) to view all of the data.

There are 4 data files contained in this export:

sites.txt - Contains data pertaining to the sites tracked in GeoTracker
regulatory_activities.txt - Contains regulatory activities for sites tracked in GeoTracker
status_history.txt - Contains the status history for sites tracked in GeoTracker
contacts.txt - Contains the regulatory contact information for sites tracked in GeoTracker

The 4 tables can be linked using the GLOBAL_ID field. The data can be opened in Excel using the steps outlined below:
1. Within Excel, click on the Open tool on the toolbar. Excel displays the Open dialog box. 
2. Using the Files of Type drop-down list at the bottom of the dialog box, indicate that you want to open Text Files (*.prn; *.txt; *.csv). 
3. Select the file you want to open. 
4. Click on Open. Excel starts the Text Import Wizard, displaying the Step 1 of 3 dialog box. 
5. Make sure the Delimited choice is selected, then click on Next. Excel displays the Step 2 of 3 dialog box. 
6. Make sure Tab is selected as a delimiter, then click on Next. Excel displays the Step 3 of 3 dialog box. The data in your TXT file should be displayed at the bottom of the dialog box. 
7. Click on Finish. Your file should be imported.
